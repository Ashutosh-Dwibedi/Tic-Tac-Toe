<h1>Tic Tac Toe</h1>

This is an offline android game project developed using Andoid Studio, Java.
<br>
<h2>Features</h2>
<ul>
  <li>Offline</li>
  <li>Supports AI mode</li>
  <li>Supports Multiplayer (Offline) mode</li>
</ul>
